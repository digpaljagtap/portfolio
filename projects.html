Drone-Based Survey and Mapping for Bridge Construction

Developed a drone-based system to survey and map bridge structures for construction management and structural health monitoring. Used UAVs with LiDAR and high-resolution cameras for topographic mapping, defect detection (cracks, corrosion), and real-time structural monitoring. Implemented data processing in Python with OpenCV, and used ArduPilot for drone calibration.

Yes Bank Stock Closing Price Prediction

Built a machine learning model to predict the monthly closing stock price of Yes Bank using historical data (Open, High, Low, Close). Analyzed the impact of the 2018 fraud case on market trends. Implemented and compared Linear Regression and Random Forest, with Linear Regression showing the best performance. Future scope includes time series models for improved forecasting.

Developed a deep learning-based system to classify brain MRI scans into distinct tumor types. Implemented both a custom CNN and transfer learning models (MobileNetV2, ResNet50, EfficientNetB0) for accurate diagnosis. Built an AI-powered system to classify fish species using both a custom CNN and transfer learning models (VGG16, ResNet50, InceptionV3, EfficientNetB0, MobileNetV2). Deployed the final model in a Streamlit web app for real-time fish species prediction with confidence scores.
